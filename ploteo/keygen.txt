1. Generar la llave (Forzando el nombre correcto)
Copia y pega este comando tal cual. No le cambies el nombre ni le pongas contraseña (solo dale Enter cuando te pregunte cosas):

Bash

ssh-keygen -t rsa -b 4096 -f ~/.ssh/id_rsa -N ""
2. Enviar la llave al servidor
Ahora debemos "presentar" esta nueva llave al servidor para que nos deje pasar:

Bash

ssh-copy-id sysop@10.54.217.146
(Aquí te pedirá la contraseña de sysop por última vez).

3. Ajustar los permisos de seguridad
SSH ignorará la llave si los permisos no son estrictos. Ejecuta esto para asegurar que solo tu usuario pueda leerla:

Bash

chmod 700 ~/.ssh
chmod 600 ~/.ssh/id_rsa
chmod 644 ~/.ssh/id_rsa.pub
